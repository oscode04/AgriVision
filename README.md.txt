AgriVision adalah aplikasi yang dikembangkan untuk membantu para petani dalam mengolah lahan pertaniannya. Fitur utama dari aplikasi ini adalah fitur rekomendasi pupuk, fitur ini dapat membantu para petani dalam memilih jenis pupuk yang tepat berdasarkan karakteristik lahan pertaniannya. Dengan memanfaatkan teknologi Machine Learning, aplikasi ini mampu untuk memberikan rekomendasi pupuk yang optimal sehingga dapat meningkatkan hasil panen dan meminimalkan biaya operasional.
